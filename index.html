O que é o Token Amazônia

O Token Amazônia é um projeto de transparência ambiental que utiliza dados públicos de satélite para monitorar, em escala continental, o estado ecológico da floresta amazônica. Este site funciona como a vitrine técnica e institucional do sistema.

Como funciona

Dados do programa Sentinel-2 são processados continuamente por meio do Google Earth Engine. A partir desses dados, é gerado um índice ecológico sintético que representa a atividade biológica da Amazônia ao longo do tempo.
